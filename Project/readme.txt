steps:
1) pull project from git
2) open command prompt and change to directory to Tracks\Project (directory containing manage.py)
3) run the server
4) point browser (preferably Chrome) to http://127.0.0.1:8000/Tracks/
5) go to http://127.0.0.1:8000/Tracks/register to register an account
	5.1) or go to http://127.0.0.1:8000/Tracks/signin to signin using an existing account
6) add track by using "choose file" and then upload
	6.1) successful file upload will show green progress bar.
	6.2) unsuccessful file upload will show red progress bar.
		6.2.1) can be unsuccessful for reasons including: file too large (>2MB), file not an mp3.

7) hit browser refresh to see updated changes.
8) create collaboration by pressing "Collaborate"
9) refresh page to see updated changes.
10) edit profile by using "edit profile" hyperlink
